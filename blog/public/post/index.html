Giới thiệu về Java

Java là một ngôn ngữ lập trình phổ biến, được sử dụng rộng rãi trong phát triển phần mềm, ứng dụng di động, và các hệ thống lớn. Java được phát triển lần đầu tiên bởi James Gosling và Mike Sheridan tại Sun Microsystems vào những năm 1990.

Trong bài viết này, chúng ta sẽ tìm hiểu về Java, các tính năng chính và lý do tại sao Java lại trở thành một trong những ngôn ngữ lập trình phổ biến nhất trên thế giới.

Các thành phần cơ bản một chương trình java

Một chương trình Java cơ bản bao gồm các thành phần sau:

  • Class: Mọi chương trình Java đều phải có ít nhất một lớp.
  • Main Method: Phương thức main() là điểm khởi đầu của chương trình.

Dưới đây là một ví dụ đơn giản về cấu trúc của một chương trình Java:

public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

Java cung cấp nhiều kiểu dữ liệu cơ bản như:

Ví dụ về cách sử dụng các kiểu dữ liệu:

Xử lý ngoại lệ trong Java

Xử lý ngoại lệ là một phần quan trọng trong lập trình Java. Nó giúp chương trình của bạn tiếp tục chạy mặc dù có lỗi xảy ra.

  • try-catch: Cấu trúc cơ bản để xử lý ngoại lệ.

    int result = 10 / 0;  // gây ra lỗi chia cho 0
} catch (ArithmeticException e) {
    System.out.println("Lỗi: " + e.getMessage());

Truyền API trong Java

Truyền API trong Java là một công cụ mạnh mẽ để xử lý các tập dữ liệu lớn theo cách hàm, hỗ trợ các thao tác như lọc, sắp xếp, và chuyển đổi mà không thay đổi dữ liệu gốc.

  • Khởi tạo Stream từ Collection:
import java.util.Arrays;
import java.util.List;

public class Main {
    public static void main(String[] args) {
        List<String> languages = Arrays.asList("Java", "Python", "C++", "JavaScript");
        
        languages.stream()
                 .filter(language -> language.startsWith("J"))
                 .forEach(System.out::println);
    }
}

Làm việc với ArrayList trong Java

ArrayList là một trong những lớp của Java cung cấp một cách tiếp cận động đối với mảng. Nó có thể thay đổi kích thước khi bạn thêm hoặc xóa phần tử.

  • Khởi tạo ArrayList:
import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList<String> list = new ArrayList<>();
        list.add("Java");
        list.add("Python");
        list.add("JavaScript");
        
        System.out.println(list);
    }
}
