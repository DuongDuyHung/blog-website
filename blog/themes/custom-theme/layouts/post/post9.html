Làm việc với ArrayList trong Java

ArrayList là một lớp trong Java cung cấp một cách tiếp cận động đối với mảng. Bạn có thể thay đổi kích thước của nó khi thêm hoặc xóa các phần tử.

Dưới đây là ví dụ về cách sử dụng ArrayList trong Java:


import java.util.ArrayList;

public class Main {
    public static void main(String[] args) {
        ArrayList numbers = new ArrayList<>();
        numbers.add(1);
        numbers.add(2);
        numbers.add(3);
        System.out.println(numbers);
    }
}
