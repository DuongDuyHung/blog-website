Java cung cấp nhiều loại vòng lặp để xử lý các tình huống lặp lại. Dưới đây là một số ví dụ:

- **for loop**: Dùng khi bạn biết số lần lặp.
  
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}
