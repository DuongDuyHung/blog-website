Java là một ngôn ngữ hướng đối tượng, vì vậy mọi thứ trong Java đều là một lớp hoặc đối tượng. Một lớp là một bản thiết kế cho đối tượng.

- **Định nghĩa lớp**:
  
```java
class Car {
    String model;
    int year;
    
    public void drive() {
        System.out.println("The car is driving");
    }
}
